Working Progress

There were some major and minor problems in the pre-made page. Firstly, it might seem random, but it isn't. It has kind of cycle because it shows a fixed image's specific part and just keep moving on the image to show diferrent visual. So I needed to think of way to vary the base image. My solution was linking random images from https://source.unsplash.com where you can literally get a random web image within a single URL. (Iframe is prohibited)

Secondly, the script requires two images: original one and flipped one. Also, it seemed to take only data-format, (I mean base-64 jpg) so I tried to put encorder & flipper to input two images.
